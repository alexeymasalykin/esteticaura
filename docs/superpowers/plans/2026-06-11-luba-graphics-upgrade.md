## Решения

- Морф-инвариант: при `uProgress=1` все частицы на месте (`aDelay∈[0,0.65]`,
  окно 0.35). GSAP по-прежнему анимирует один scalar — стаггер в шейдере.
- Уход бриллианта анимируется по `points.position/scale` + `uOpacity` —
  обратим скроллом (scrub), rotation.y остаётся покадровым.
- Рёберные частицы сэмплируются по длине рёбер треугольников (уникальные
  пары индексов), доля ~28% бюджета.
