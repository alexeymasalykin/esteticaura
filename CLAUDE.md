## Жёсткие правила (не нарушать)
1. **Контраст WCAG.** Золото `#D4AF37` текстом — ТОЛЬКО для крупного (заголовки/цифры, ≥3:1).
   Body и подписи — `--text`/`--text-muted`, никогда золотом. Body ≥ 4.5:1.
2. **reduced-motion.** При `prefers-reduced-motion` — НЕ грузить Three.js (статичный постер),
   отключить ScrollTrigger-хореографию и параллакс. Hover-переходы оставить.
3. **Перформанс.** Three.js не влезает в бюджет лендинга (<150kb JS) — осознанно. Митигация:
   динамический импорт 3D после первого рендера; на mobile/слабом GPU — постер вместо сцены.
4. **Touch ≥ 44px**, форма: `type="tel"`, `autocomplete`, валидация на blur, `aria-live`.
5. **Числа** (цены, мета-цифры) — `font-variant-numeric: tabular-nums`.
6. **Никаких inline-стилей** в `index.html` — только классы дизайн-системы.

## 3D-параметры (из борда — стартовые)
Капля: `MeshPhysicalMaterial { transmission:1, thickness:0.5, roughness:0.05, ior:1.33, iridescence:0.2 }`
(roughness 0.2–0.6 пикселит — избегать). Кристалл: `metalness:1, roughness:0.15, flatShading`.
Env: `RoomEnvironment` + `PMREMGenerator` (без HDRI-файлов). Bloom: `RenderPass → UnrealBloomPass(0.4,1.2,0.1) → OutputPass`
(**OutputPass обязателен последним** — иначе ломает тон-маппинг). Селективный bloom НЕ нужен (UI вне WebGL).
